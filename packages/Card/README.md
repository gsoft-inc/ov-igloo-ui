# @igloo-ui/card

Welcome to my `Card` component.

## Installation

```sh
npm i @igloo-ui/card
# or with yarn
yarn add @igloo-ui/card
```

Then to use the component in your code just import it!

```js
import Card from '@igloo-ui/card';
// and with css-modules
import '@igloo-ui/card/dist/card.css';
```

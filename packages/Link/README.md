# @igloo-ui/link

Welcome to my `Link` component.

## Installation

```sh
npm i @igloo-ui/link
# or with yarn
yarn add @igloo-ui/link
```

Then to use the component in your code just import it!

```js
import Link from '@igloo-ui/link';
// and with css-modules
import '@igloo-ui/link/dist/link.css';
```

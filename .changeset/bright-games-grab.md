## Ellipsis component

Add Ellipsis component to allowing text truncation in a div.

### Component API

| name     | description                                                    |
| -------- | -------------------------------------------------------------- |
| children | The content to display inside the component                    |
| title    | The title to display when hovering the mouse over the ellipsis |
